# Three.js - Transformaciones

Aplicación web desarrollada con React Three Fiber para demostrar transformaciones geométricas (traslación, rotación, escala) sobre objetos 3D interactivos.

## Ejecución

```bash
npm install
npm run dev
```

## Características
- Renderizado de un Icosaedro interactivo utilizando `meshStandardMaterial`.
- Uso del hook `useFrame` para animar continuamente la rotación, posición y escala.
- Interfaz integrada con `leva` para ajustar las propiedades de transformación en tiempo real.
