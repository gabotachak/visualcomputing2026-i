# Repositorio de Talleres de Computación Visual 2026-I

Este repositorio contiene las soluciones de los talleres prácticos del curso de Computación Visual.

**Estudiante:** Gabriel Andres Anzola Tachak
**Departamento:** Ingeniería de Sistemas e Industrial - Universidad Nacional de Colombia
**Semestre:** 2026-I

---

## Descripción del Curso

Este curso proporciona una introducción a los conceptos básicos de la computación gráfica y el procesamiento de imágenes, tomando como eje central el estudio del pipeline gráfico presente en diferentes plataformas modernas. Se exploran tanto aspectos teóricos como prácticos mediante implementaciones en múltiples entornos (Unity, Three.js, Python).
