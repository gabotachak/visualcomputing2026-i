# Media

En este directorio se almacenan los archivos multimedia (capturas de pantalla, *screencasts* en GIF y videos) generados durante la ejecución de los experimentos y aplicaciones del taller `semana_01_3_jerarquias_transformaciones`.

## Estructura

- `/screenshots/`: Grabaciones de pantalla de la interacción completa con las herramientas (e.g `threejs.gif`).
- `/threejs/`: Capturas estáticas tomadas directamente desde del canvas de React Three Fiber de las implementaciones. 
- `/unity/`: (Reservada para futuras implementaciones de Unity)
